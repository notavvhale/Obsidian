1. ПК
	1.  Создать учётку на https://github.com/
	2.  Создать репозиторий для базы. В гайде написано поставить Private но у меня возникла проблема с клонированием приватного репозитория на телефоне, поэтому я оставил паблик.![[Pasted image 20260227154723.png]]
	3. Установить git https://git-scm.com/
	4. Сгенерировать GitHub Токен (необязательно). Я так понял он нужен для авторизации автоматической. НО у меня он не используется никак, поэтому описывать не буду. Можно посмотреть в гайде. SSH Authentication называется.
	5. Сгенерировать GitHub Personal Access Token. Это универсальная замена паролю. Почему-то по паролю не давало склонировать репозиторий из Obsidian. Работает так что каждому токену можно задать определённые права. Я создал токен и все права поставил write&read. Чтобы создать токен нужно перейти в параметры справа сверху > Developer settings > Personal access tokens > Tokens (classic) > Generate new token > Generate new token (classic).![[Pasted image 20260227173918.png]]
	   Дальше нужно написать название токена и выбрать права который он будет давать. Я выбрал все. И ещё надо поставить чтобы токен был бесконечным.![[Pasted image 20260227174122.png]]
	6. Дальше переходим в Obsidian, устанавливаем плагин Git. Чтобы установить плагин нужно перейти в Параметры > Плагины Сообщества (Settings > Community Plugins). В параметрах ставить auto commit and sync. Чтобы не было конфликтов между версиями надо включить Pull on Startup.![[Pasted image 20260227174651.png]]![[Pasted image 20260227174713.png]]
	7. Далее нужно склонировать репозиторий средствами плагина Obsidian. В гайде написано, что клонировать репозиторий нужно ручками через Powershell, но если так делать, то плагин почему-то через раз определяет этот склонированный репозиторий. Поэтому жмём "палитра команд" и ищем команду Clone existing repo.![[Pasted image 20260227174917.png]]![[Pasted image 20260227174920.png]]
	8. После клонирования проверяем, что плагин корректно склонировал репозиторий куда надо. Должно выглядеть так:![[Pasted image 20260227175151.png]]
	9. После этого открываем панель управления и жмём Commit and sync.![[Pasted image 20260227175315.png]]![[Pasted image 20260227175413.png]]
	10. После этого должно всё зафетчится и закомитится и запушится одним словом должно заработать. Если не заработало, как и у меня, то качаем Github Desktop и жмём Fetch или какую-то кнопку тут. И всё полетит.![[Pasted image 20260227175739.png]]
