Это служба каталогов, которая используется для хранения и управления данными объектов домена. Включает в себя логические(домен, лес) и физические(домен контроллер, сайт) компоненты. Логические описывают структуру, а физические материальные компоненты реального мира.

**Учётная запись пользователя** домена это объект, включающий в себя имя пользователя, пароль и членство в группах. Также может сдержать дополнительные параметры.

Групповые объекты.
**Локальные группы** - доступны только на станциях, где они существуют. Права группы только для локальных ресурсов. Участники могут быть из любой точки леса AD DS.
**Локальные для домена** - права группы применяются в любой точке леса AD DS. Группа может включать только объекты локального домена.
**Глобальные группы** - Используется для объединения пользователей со схожими характеристиками, например с одного отдела. Права группы применяются в любой точке леса. Может содержать только объекты локального домена.
**Универсальные группы** - используется в многодоменных сетях. Права применяются так же как у глобальных. Объекты могут быть из любой точки леса.

###### Компьютерные объекты.
Имеют учётную запись с именем пользователя и паролем, которые Windows автоматически периодически меняет. Проходят аутентификацию в домене. Могут входить в группы, иметь доступ к ресурсам а также могут быть настроены с помощью групповой политики.

###### Контейнер с компьютерами
Это объект класса Container. Не является организационным подразделением OU.
По дефолту в этот контейнер помещаются объекты учётных записей компьютеров, добавленных в домен. На этот контейнер не будут применяться групповые политики. Нельзя создавать OU внутри контейнера. **Microsoft рекомендует создавать отдельные OU для размещения этих объектов вместо использования контейнера по дефолту.**

**Лес и домен AD DS**
Домен - логическая зона управления в Active Directory.
Лес - самая большая граница управления. Включает в себя один или несколько доменов, общий глобальный каталог.

Abatum.com - отдельный домен в лесу.
Contoso.com - корневая область леса.
Seattle.contoso.com - дочерний элемент Contoso.

**Что такое MSA / gMSA / dMSA**
Это всё — **служебные учётные записи (Service Accounts)** в Active Directory, которые используются для запуска сервисов.
