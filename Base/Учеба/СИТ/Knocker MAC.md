нокер на маке

Создаем текстовый документ через програму textedit
Конвертируем документ в обычный текст 
Вставляем нокер


#!/usr/bin/env bash
ping interlog.vl-911.ru -s 274 -c 1
ping interlog.vl-911.ru -s 55 -c 1
ping interlog.vl-911.ru -s 904 -c 1
open /Applications/"Microsoft Remote Desktop.app"
killall Terminal
exit

Стираем энтер и вводим заново энтер(хз ричард писал я не понял что это не понадобилось)
сохраняем файл с расширение .sh(можно сохранить в любом формате, а потом переименовать файл. У меня в .sh не открывалось переименовал в .command)
Возможно нужно будет удалить первую строчку в нокере(тоже хз не пригодилось так работает)
Правой кнопкой мыши, всплывающее меню, открыть с помощью, терминал, установить по умолчанию (.command если .sh не открывается)

для записи в файл hosts
sudo nano /etc/hosts
Сохранить изменения и закрыть редактор. В nano это делается сочетанием клавиш Ctrl+O, затем Enter и Ctrl+X для выхода.

Файлик открывает Microsoft Remote Desktop
Его нужно загрузить с AppStor'a
Файл должен оставаться .command  правой кнопкой мыши и (rename)
Изменяется обычным текстовиком TextEdit
нужно дать права файлу chmod 777 ФАЙЛ(назначить исполняемым)(открыть терминал, cd до места где лежит файлик и написать
chmod 777 knocker.command)
после этого нужно дать ещё какието права командой 
xattr -d com.apple.quarantine knocker.command
Ещё нужно убрать символы /r если скопирован файл с винды, командой 
sed 's/\r$//' knocker.command > knocker1.command



Что-бы заменить иконку на другую, заходим в свойства файла, в левом верхнем углу видим его значок
выбираем любую картинку(хоть с браузера) и перетаскиваем мышкой на иконку нашего файла и вуаля она меняется
